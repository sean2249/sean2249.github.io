    How to scale your web and mobile applications

    五月 6, 2022 分類 linkPage

    https://enlear.academy/how-to-scale-your-web-and-mobile-applications-5be74bf99226

    • 從最簡單的 web app，因應服務人數的升級及 SLA 的要求，不斷的調整架構

    1. Basic design for a web app

      • single app server
      • Database

    2. Scaling the app servers: requests to the application server grow. Need for leveling up the CPU capacity.

      • Load balancers for multiple app server
      • Session store: memcached/redis

    3. Scaling the database with caching: database crisis for incread requests. store most requested data in the distributed cache system

      • distributed cache system to support database

    4. Distributing the database: caching is not enough to reduce the volume of work on your databse -> horional scaling (stored across multiple DB instances)

      • Distributed database system

    5. Increasing RESPONSIVENESS: Some mission takes lots of time, and cannot response in time.

      • queuing system

    6. we started from the design of a simple Web app

    7. we increased the compute capacity by adding more instances and a load balancer

    8. we relieved our DB from the extra load by leveraging the caching system

    9. we scaled out DB, turning it into a distributed database system

    10. we improved the responsiveness of our application using a queuing system
